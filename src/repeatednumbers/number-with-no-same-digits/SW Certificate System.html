“Number with No Same Digits” means that the numbers is made up of different digit only. 


For example, the number 55 has two digits which are the same (5), but the number 54 has two digits which are different (5 and 4). 


When given a number M of length N, find the “Number with No Same Digits” closest to the given M. If there are two, find the smaller number.


[Constraints]

1. N is an integer from 2 to 10.

2. M is an integer from 10 to 9,999,999,999. This may exceed the int (32 bits) range.


[Input]

In the first line, the number of test casts (T) is given, and then T number of test cases will be given. In the first line of each test case, N and M are given between spaces. 


[Output]

Standard print the answers of each test case in order. Each test case begins with “#x” (x is a test case number, starts from 1) and (excluding quotation marks) after a space, print the answer
