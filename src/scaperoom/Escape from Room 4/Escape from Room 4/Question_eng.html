Seung-jun(SJ) is playing an escape game in a room with a grid of H rows and W columns. 


It takes 1 unit of time for SJ to move to a neighboring space from the current space. He is not able to escape because the room is built to be inescapable. SJ, who is physically fit, couldn't stand being in the room any longer and decided to escape by breaking the wall.


Because he is physically strong, SJ can move by breaking several consecutive walls in the direction he is moving. Even if he does this, only 1 unit of time is consumed. He can stop while breaking walls, and he has to stop when there are no more walls to break.


However, breaking the walls like this consumes a lot of energy, so there is only one time that the walls can be broken. Let's write a program that outputs the minimum time it takes for SJ to travel to the exit when SJ's current location and the shape of the room are given. However, if it is impossible to escape the room, output -1.


Below is an example of 7 x 7 grid.



[Constraints]

1. H and W, the size of the room are integers between 5 and 500, including 5 and 500. 

2. The row no. R at the SJ’s initial location is an integer between 2 and H - 1, including 2 and H - 1, and the column no. C is an integer between 2 and W - 1, including 2 and W - 1.

3. The entire perimeter of the room is surrounded by walls.

4. SJ’s initial location is not a wall.

5. There is only one exit on the room.


[Input]

In the first line, the number of test cases (T) is given, and then T number of test cases will be given. In the first line of each test case, H, the height of room and W, the width of room are given. In the second line, R and C, Seung-jun(SJ)’s initial location are given. And then in the next R number of lines, the shape of the room is displayed. ‘X’ is wall, ‘_’ is an area where one can move, ‘E’ is an exit.


[Output]

Output a test case on one line per test case. For each test case, output “#x” (where x is the number of the test case, starting from 1), leave a space, and then output the answer of the relevant test case.
