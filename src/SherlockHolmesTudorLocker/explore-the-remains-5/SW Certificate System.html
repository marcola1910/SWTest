Historian Sherlock Holmes and his assistant Watson have been exploring a historic site for a long time. After a long search for remains, they discovered a treasure chest with an old book about the Tudor dynasty inside. But the treasure chest is locked.


The lock has a dial and some buttons. As shown in [Figure 1], the dial has M graduations, marked with integers ranging from 0 to M-1 in a clockwise direction. The starting number is 0. Dial has N number of buttons. The dial moves only when you push the buttons, and it moves clockwise by the number of spaces equal to the number on the pushed button.


For instance, if you press a button with number 6, the dial would move 6 spaces clockwise and end up pointing at 2 as shown in [Figure 2]. If you press a button with number 5, then the dial would point at 3 after rotating 5 spaces clockwise as displayed in [Figure 3].




Having studied the Tudors for a long time, Sherlock found out how to release the dial lock which has been passed down through only a selected few people of the royal family. There is a selected number, K, inside the lock. If the dial points at K by pressing the buttons N-1 times at maximum, then the box opens!


When Sherlock was about to press the buttons, Watson wanted to tease Sherlock and threw a question. “Hey Sherlock, how many ways do you think exist to release the lock by pressing buttons up to N-1 times? It’d be impossible to work this out in your head even for a smart person like you, right?” Sherlock, a man of high self-respect, felt like he would not be able to open the chest until he answers Watson’s question. 


Let’s help Sherlock find the number of ways he can open the chest.



(Explanation to Exercise 1)
There are a total of 21 ways as shown below, to make the dial point at 3 by pressing the buttons.

[2, 2, 7], [2, 5], [2, 6, 7], [2, 7, 2], [2, 7, 6], [5, 2], [5, 6], [5, 5, 5], [5, 7, 7] [6, 2, 7], [6, 5], [6, 6, 7], [6, 7, 2], [6, 7, 6], [7], [7, 2, 2], [7, 2, 6], [7, 5, 7], [7, 6, 2], [7, 6, 6], [7, 7, 5]



[Restrictions]
1. The number of buttons, N, is an integer ranging from 4 to 8.
2. The number of dial graduations, M, is an integer ranging from 2 to 100.
3. The condition for releasing the dial lock, K, is an integer ranging from 0 to M-1.
4. The numbers on the buttons are integers ranging from 1 to 100000.
5. You must press the button at least 1 time; you can press them up to N-1 times.
6. You can press the same button that you had already pressed previously.
7. No same number can be on the buttons.
8. Even if the dial points to K while pressing the buttons, if the dial points to K by pressing the buttons additionally, it is considered a different method.


[Input]
The first line of the input file contains T, the number of test cases.
Each test case consists of two lines. The first line of each test case provides the number of buttons N, the number of dial graduations M, and the condition for dial lock release K. The second line provides N numbers carved on the buttons; the numbers are separated by a blank.


[Output]
Print the answers to each test case. Each line shall begin with #x (x is the test case, starts from 1).  Then, put in a blank and print the number of ways to make the dial point at K by pressing the button up to N-1 times.
